1. Coordenadas de un Vector que Une Dos Puntos

Supongamos que tenemos dos puntos A(x₁, y₁) e B(x₂, y₂). El vector AB se calcula restando a las coordenadas del punto de destino (final) las coordenadas del punto de origen (inicial):

⃗AB = ⃗OB − ⃗OA = (x₂ − x₁, y₂ − y₁)
Ejemplo: Dados M(7, −5) y N(−2, −11):
⃗MN = N − M = (−2 − 7, −11 − (−5)) = (−9, −6)
⃗NM = M − N = (7 − (−2), −5 − (−11)) = (9, 6)

2. Condición para que Tres Puntos estén Alineados

Para que tres puntos A, B y C estén alineados, los vectores ⃗AB y ⃗BC tienen que tener la misma dirección. Es decir, sus coordenadas tienen que ser proporcionales:

(x₂ − x₁) / (x₃ − x₂) = (y₂ − y₁) / (y₃ − y₂)
Ejemplo: ¿P(7,11), Q(4,−3) y R(10,25) están alineados?
⃗PQ = (−3, −14) y ⃗QR = (6, 28). Como ⃗QR = −2 · ⃗PQ (son proporcionales), SÍ están alineados.

3. Punto Medio de un Segmento

Las coordenadas del punto medio M de un segmento de extremos A(x₁, y₁) y B(x₂, y₂) se obtienen calculando la media aritmética de las coordenadas de sus extremos:

M = ( (x₁ + x₂)/2 , (y₁ + y₂)/2 )
Ejemplo: Dados A(2, 3) y B(4, 5):
M = ( (2+4)/2 , (3+5)/2 ) = (6/2, 8/2) = (3, 4)
